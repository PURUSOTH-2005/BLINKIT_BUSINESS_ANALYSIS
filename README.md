# BLINKIT_BUSINESS_ANALYSIS-
This project analyzes Blinkit’s business datasets (customers, orders, products, marketing, and inventory) to uncover actionable insights and build interactive dashboards. The goal is to demonstrate end‑to‑end data preprocessing, analysis, and visualization using Excel, SQL, Python, and Power BI.

## 📌 Overview
This project analyzes **Blinkit’s business datasets** (customers, orders, products, marketing, and inventory) to uncover actionable insights and build interactive dashboards. It demonstrates end-to-end **data preprocessing, analysis, and visualization** using **Excel, SQL, Python, and Power BI**.

---

## 🎯 Objectives
- **Customer Insights** → Segment customers by activity, average order value, and feedback sentiment.  
- **Order Trends** → Track monthly orders, revenue growth, and delivery performance.  
- **Marketing ROI** → Evaluate campaign conversions, return on ad spend (ROAS), and channel effectiveness.  
- **Inventory KPIs** → Monitor stock levels, damaged stock trends, and product shelf life.  

---

## 🛠️ Workflow
1. **Data Preprocessing**  
   - Cleaned raw CSV files  
   - Handled missing values  
   - Encoded categorical variables  
   - Standardized numerical features  

2. **Exploratory Data Analysis (EDA)**  
   - Correlation analysis between customer segments, order frequency, and product categories  
   - Visualizations using Python (Matplotlib, Seaborn)  

3. **SQL Queries**  
   - Extracted KPIs such as top-selling products, high-value customers, and campaign performance  

4. **Power BI Dashboard**  
   - Built interactive dashboards with slicers, KPIs, and visualizations for decision-making  

---

## 📊 Key Insights
- Premium customers showed the **highest average order value (₹1,100+)**  
- Feedback analysis revealed **34% neutral sentiment**, highlighting scope for service improvement  
- Marketing campaigns achieved **ROAS ~2.75**, with app and email channels performing best  
- Inventory analysis identified **high damage rates in household care products**, guiding supply chain optimization
